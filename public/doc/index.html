Get started

The Café API allows you to create, update, retrieve and delete products from our webstore - Phonecafé.

Base URls
http://localhost:3000/products (When running on your computer)
http://cafe-api.corentin.codes/products (From anywhere)

The Product Object


 # Example Product
 {
     "id": 17940,
     "brand": "Apple",
     "model": "iPhone X",
     "os": "iOS",
     "image": "https://i.imgur.com/Xr4H3Hu.png",
     "screensize": 5
 }
Field Type Description
brand String Product's brand
model String Product's model name
os String Product's Operating System
screensize Integer Product's Screen Size
image String Image of the product

Retrieve Products


 # JavaScript Example (jQuery is required)
 const data = {
     brand: 'Apple',
     model: 'iPhone',
     screensize: 7
 }
 $.get('http://localhost:3000/products', data, function(products) {
     products.forEach(function (product) {
         console.log(product.brand)
     })
  });
                 
                 
 # cURL Example
 curl \
 -X GET http://localhost:3000/products \
 -F 'brand=Apple' \
 -F 'model=iPhone' \
 -F 'screensize=7'

Query Parameters

Field Type Required
brand String Optional
model String Optional
os String Optional
screensize Integer Optional
image String Optional

If no query are specified, all products will be returned.


Returned HTTP Codes

Code Meaning
200 The request has succeed. Please notice that the data can be empty.
500 An Error occured. It mainly occurs when some parameters are missing or incorrect.

 # cURL Example
 curl \

GET/{:id}

Provided ID must a valid Integer. If whether it's not or no product belongs to this ID, nothing will be returned.

Returned HTTP Codes

Code Meaning
200 The request has succeed. Please notice that the data can be empty.
500 An Error occured. It mainly occurs when some parameters are missing or incorrect.

Create Product


 # JavaScript Example (jQuery is required)
 const data = {
     brand: 'Apple',
     model: 'iPhone',
     screensize: 7
 }
 $.ajax({
     type: "POST",
     url: "http://localhost:3000/products",
     data: JSON.stringify(data),
     contentType: "application/json"
 }).done(function(data) {
     // Product has been created
 }).fail(function(error) {
     // An error occurred
 }); 
                 
                 
 # cURL Example
 curl \
 -X POST http://localhost:3000/products \
 -H "Content-Type: application/json"
 -D '{ "brand": "Apple", "model": "iPhone", "os": "iOS 10", "image": "https://i.imgur.com/Xr4H3Hu.png", "screensize": 7}'

POST/

The following parameters must be provided by JSON-Encoded data.


Body Parameters

Field Type Required
brand String Required
model String Required
os String Required
screensize Integer Required
image String Required

If no query are specified, all products will be returned.


Returned HTTP Codes

Code Meaning
201 The Product has been created.
500 An Error occured. It mainly occurs when some parameters are missing or incorrect.

Update Product


 # JavaScript Example (jQuery is required)
 const data = {
     screensize: 8,
     brand: 'Google'
 }
 $.ajax({
     type: "PUT",
     url: "http://localhost:3000/products/56",
     data: JSON.stringify(data),
     contentType: "application/json"
 }).done(function(data) {
     // Product has been created
 }).fail(function(error) {
     // An error occurred
 }); 
                 
                 
 # cURL Example
 curl \
 -X POST http://localhost:3000/products/56 \
 -H "Content-Type: application/json"
 -D '{ "screensize": 7, "brand": "Google" }'

PUT/{:id}

The following parameters must be provided by JSON-Encoded data. You can chose which of them by providing only the ones that you want to update.

The example request only updates the fields `screensize` and `brand`.


Body Parameters

Field Type Required
brand String Optional
model String Optional
os String Optional
screensize Integer Optional
image String Optional

If no query are specified, the product will not be updated


Returned HTTP Codes

Code Meaning
201 The Product has been created. Please notice that the request will return 201 even if no products were found.
500 An Error occured. It mainly occurs when some parameters are incorrect.

Delete Product

                     
 # JavaScript Example (jQuery is required)
 $.ajax({
     type: "DELETE",
     url: "http://localhost:3000/products/65"
 }).done(function(data) {
     // Successfully deleted product
 }).fail(function(error) {
     // An error occurred
 });
                     
                     
 # cURL Example
 curl \
 -X DELETE http://localhost:3000/products/65

DELETE/{:id}

Provided ID must a valid Integer. If it's not, an error will happen.

Returned HTTP Codes

Code Meaning
201 The request has succeed. Please notice that the request will return 201 even if no products were found.
500 An Error occured. It mainly occurs when the provided ID was incorrect.
